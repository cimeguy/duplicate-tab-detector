# 重复标签检测 · Duplicate Tab Detector

一个轻量的 Chrome 扩展，自动检测重复标签页并支持按网站归类标签组。

## 功能

- **页面顶部横幅**：重复标签在页面顶端显示橙色提示条 `⚠ 此页面已重复打开 N 次`
- **扩展图标 Badge**：右上角实时显示当前重复组数
- **弹出面板**：点击扩展图标，列出所有重复组及其标签标题，点击可直接跳转
- **按标题归类标签**：一键将当前窗口中同一网站的标签归入 Chrome 原生标签组，支持预览确认后再执行
- **精准判断**：完整 URL（含路径、查询参数、Hash）完全一致才视为重复，参数不同的页面不会被误判
- **实时更新**：标签新建、关闭、导航时自动重新检测

## 安装

1. 下载或克隆本仓库到本地
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 右上角开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本仓库目录
5. 扩展即刻生效

## 使用说明

### 重复标签检测
打开多个相同 URL 的标签页后，每个重复标签顶部会自动出现橙色横幅提示。点击扩展图标可在弹出面板中查看所有重复分组，点击任意行即可跳转到对应标签。

### 按标题归类标签
点击弹出面板中的「🗂 按标题归类标签」按钮，扩展会分析当前窗口中同一网站（根据标签标题末尾的网站名识别）是否有 2 个以上标签，并显示预览。确认后，这些标签将被归入以网站名命名的 Chrome 标签组，并自动分配不同颜色。
