# 重复标签检测 · Duplicate Tab Detector

一个轻量的 Chrome 扩展，自动检测并高亮重复打开的标签页。

## 功能

- **页面顶部横幅**：重复标签在页面顶端显示橙色提示条 `⚠ 此页面已重复打开 N 次`
- **扩展图标 Badge**：右上角实时显示当前重复组数
- **弹出面板**：点击扩展图标，列出所有重复组及其标签标题，点击可直接跳转
- **精准判断**：完整 URL（含路径、查询参数、Hash）完全一致才视为重复，参数不同的页面不会被误判
- **实时更新**：标签新建、关闭、导航时自动重新检测

## 安装

1. 下载或克隆本仓库到本地
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 右上角开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本仓库目录
5. 扩展即刻生效

## 截图

> 重复标签页面顶部会出现橙色提示横幅，点击扩展图标可查看所有重复列表并跳转。
